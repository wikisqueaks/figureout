# Set Default Journal

Sets a global option for the target journal. This prevents you from
needing to specify the `journal` argument in every call to
[`publish_fig()`](https://wikisqueaks.github.io/figR/reference/publish_fig.md).

## Usage

``` r
set_journal(journal)
```

## Arguments

- journal:

  Character. Name of the journal (e.g., "Nature").
